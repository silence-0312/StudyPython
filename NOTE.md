# 1, 面向对象概述（objectOriented, OO）
- OO思想
    - 接触到任何一个任务，首先想到是这个任务所组成的物件是由模型构成
- 介绍名词
    - OO：面向对象
    - OOA:对面对象的分析
    - OOD:对面对象的设计
    - OOI:对面对象的实现
    - OOP:对面对象的编程
    - OOA ->OOAD ->OOI:面向对象实现的过程
-  类和对象的概念
    - 类：楚抽象名词，代表一个集合，一个共性的事物
    - 对象：一个具象的事物，单个个体
    - 类中的类容，应该具有两个内容
        -   表明事物的特征，叫做属性（变量）
        -   表明事物的功能和动作，称为成员方法（函数）
